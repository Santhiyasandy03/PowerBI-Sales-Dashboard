# Sales Performance Dashboard

## Overview
Interactive Power BI dashboard created to analyze sales performance, profitability, regional trends, and product categories.

## Tools Used
- Power BI
- DAX
- Excel

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## Dashboard Features
- Monthly Sales Trend Analysis
- Regional Sales Performance
- Category-wise Sales Distribution
- Interactive Filters and Slicers

## Key Insights
- Sales performance varied across regions and product categories.
- Certain product categories contributed a larger share of revenue.
- Regional trends highlighted business growth opportunities.
